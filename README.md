# Code Test Bot App

The Ember.js and Rails front-end for the Code Test Bot application.

### Installing
##### System Dependencies
- Ruby 2.1.1


##### Bundler
Use Bundler to install Gem dependencies.
```sh
bundle install
```


##### Running Tests
Use teaspoon as a test runner to execute the JS tests using PhantomJS.
```sh
rake teaspoon
```
